Overscore headings

The overscore class adds a fixed-width line above the heading and makes the text match bodytext. It is designed to be used on paragraphs, but can be extended to headings as long as type-etc-regular or type-etc-bold classes are used. It is sometimes combined with the text-uppercase and type-etc-bold classes, to make it more heading-like.
