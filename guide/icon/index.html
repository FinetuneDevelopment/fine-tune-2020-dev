Icons

CSS Shapes

CSS take on whatever the current text colour happens to be. They need to be used along with the shape class.

Arrows

Other

ic-arrow is used in the marquee and has to be built differently as the marquee uses text-shadow to create the effect of multiple strings of text. CSS shapes do not duplicate using this technique.

Icon sizing

Large

Large icons have a container of 63px. The icons themselves inside are 35px. The first use is on the about us page. They can be applied using a icon-large class.

Small

Small icons have a container of 38px. The icons themselves inside are 15px. The first use is on insights pages, on the right hand side, for social media icons. They can be applied using a icon-small class.

Tiny

Tiny icons have a container of 32px. The icons themselves inside are 12px. They are used in the bottom right hand corner of the footer, for social media icons. They can be applied using a icon-tiny class.
